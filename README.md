## 特性
- 做你的健康小督查
- 在你忙碌的工作中提醒你喝一杯茶, 休息片刻
- 在你紧张的学习时提醒你该放下书, 爱护一下你的眼睛了
- 在你忘我的游戏时提醒你, 一切娱乐都要适度, 好好吃饭

## 编译
```bash
$ git clone https://github.com/huangcheng/ToToAo.git
$ cd ToToAo
$ qmake
$ make
```
